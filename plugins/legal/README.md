# Legal Plugin — Thai Legal Assistant

Plugin สำหรับที่ปรึกษากฎหมายไทย ตรวจสัญญา วิจัยกฎหมาย และร่างเอกสารทางกฎหมาย

## Features

### Skills (3 ทักษะ)

| Skill | Description | Routing |
|-------|-------------|---------|
| **contract-review** | ตรวจสัญญา ระบุข้อควรระวัง แนะนำการแก้ไข | reviewer, coder |
| **legal-research** | วิจัยกฎหมาย ค้นหามาตราและคำพิพากษาด้วย RAG | reviewer, orchestrator |
| **thai-law-knowledge** | ความรู้ระบบกฎหมายไทย อายุความ โครงสร้างศาล | all agents |

### Commands (4 คำสั่ง)

| Command | Usage | Description |
|---------|-------|-------------|
| `/review-contract` | `/review-contract <path>` | ตรวจสัญญาพร้อมแนะนำการแก้ไข |
| `/legal-memo` | `/legal-memo <legal-issue>` | เขียนบันทึกข้อความทางกฎหมาย |
| `/draft-demand-letter` | `/draft-demand-letter <creditor> <debtor> <amount> <due-date>` | ร่างหนังสือทวงถามหนี้ |
| `/find-laws` | `/find-laws <legal-topic>` | ค้นหากฎหมายและคำพิพากษาที่เกี่ยวข้อง |

## RAG Integration

### ใช้งาน Semantic Search

```bash
# 1. Index เอกสารกฎหมาย
> index_codebase("legal/thai-laws/")
→ Indexed 150 files, 890 chunks

# 2. ค้นหาด้วยความหมาย
> semantic_search("เลิกจ้างไม่เป็นธรรม ค่าชดเชย")
→ เจอ: พ.ร.บ. คุ้มครองแรงงาน มาตรา 118, 121
→ เจอ: ฎีกาที่ 1234/2565

# 3. ค้นหาในสัญญา
> semantic_search("ข้อบอกล่วงหน้าสัญญาเช่า")
→ เจอ: legal/contracts/lease-template.md (score: 0.89)
```

## ตัวอย่างการใช้งาน

### 1. ตรวจสัญญา

```
> /review-contract legal/contracts/lease-agreement.md

## สรุปสัญญา
- คู่สัญญา: ผู้ให้เช่า A กับ ผู้เช่า B
- วัตถุประสงค์: เช่าอาคารพาณิชย์
- ระยะเวลา: 3 ปี

## 🔴 ข้อควรระวัง (Critical)
1. Clause 5.3 — ผู้ให้เช่าริบเงินประกันทันทีหากผิดนัด
   - **ปัญหา:** เกินสมควร อาจเป็นโมฆะตาม ป.พ.พ. มาตรา 379
   - **แนะนำ:** แก้เป็น "ริบเงินประกันตามความเสียหายจริง"
```

### 2. เขียน Legal Memo

```
> /legal-memo ลูกจ้างเลิกจ้าง นายจ้างไม่จ่ายค่าชดเชย

## บันทึกข้อความทางกฎหมาย

### 1. ข้อเท็จจริง
- ลูกจ้างทำงาน 5 ปี
- นายจ้างเลิกจ้างโดยไม่บอกล่วงหน้า

### 2. ประเด็นกฎหมาย
1. ลูกจ้างมีสิทธิได้ค่าชดเชยเท่าไร

### 3. บทบัญญัติที่เกี่ยวข้อง
**พ.ร.บ. คุ้มครองแรงงาน มาตรา 118**
- ทำงาน 5 ปี ได้ 300 วัน

### 4. ข้อเสนอแนะ
1. ส่งหนังสือทวงถาม
2. ร้องเรียนสำนักงานสวัสดิการแรงงาน
3. ฟ้องศาลแรงงาน
```

### 3. ร่างหนังสือทวงถามหนี้

```
> /draft-demand-letter "นาย ก" "นาย ข" 100000 "2025-12-31"

[ร่างหนังสือทวงถามหนี้พร้อมข้อมูลครบถ้วน]

## ขั้นตอนต่อไป
1. ส่งทางไปรษณีย์ตอบรับ
2. เก็บหลักฐานการส่ง
3. หากไม่ชำระ → ฟ้องศาล
```

### 4. ค้นหากฎหมาย

```
> /find-laws เลิกจ้างไม่เป็นธรรม

## กฎหมายที่เกี่ยวข้อง

### พ.ร.บ. คุ้มครองแรงงาน มาตรา 118
- ทำงาน 120 วัน - 1 ปี: 30 วัน
- ทำงาน 1 - 3 ปี: 90 วัน
- ทำงาน 3 - 6 ปี: 180 วัน
- ทำงาน 6 - 10 ปี: 240 วัน
- ทำงาน 10 ปี+: 300 วัน

## คำพิพากษาที่เกี่ยวข้อง
- **ฎีกาที่ 1234/2565** — นายจ้างย้ายงานลูกจ้างโดยไม่ชอบ
- **ฎีกาที่ 5678/2563** — ลดเงินเดือนลูกจ้างโดยไม่ยินยอม
```

## MCP Integration

Plugin รองรับ MCP servers สำหรับเชื่อมต่อกับ:
- **Thai Law API** (กรมคุ้มครองสิทธิฯ) — ค้นหากฎหมาย
- **Supreme Court API** — ค้นหาคำพิพากษา

```json
{
  "mcpServers": {
    "thai-law-api": {
      "url": "https://api.krisdika.go.th/sse",
      "tool_filter": ["search_law", "get_judgment"]
    }
  }
}
```
