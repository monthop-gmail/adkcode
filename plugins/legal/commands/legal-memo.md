# Command: /legal-memo

## Usage
```
/legal-memo <legal-issue>
```

## Workflow

### 1. ระบุประเด็น
- ถามกลับถ้าข้อมูลไม่ชัดเจน
- แยกข้อเท็จจริง vs ประเด็นกฎหมาย

### 2. ค้นหาบทบัญญัติ
```python
semantic_search("<legal-issue> กฎหมายที่เกี่ยวข้อง")
web_search("กฎหมายไทย <legal-issue> ล่าสุด")
```

### 3. ค้นหาคำพิพากษา
```python
semantic_search("<legal-issue> คำพิพากษา ฎีกา")
```

## Example
```
> /legal-memo ลูกจ้างเลิกจ้าง นายจ้างไม่จ่ายค่าชดเชย

## บันทึกข้อความทางกฎหมาย

**วันที่:** 2026-03-06
**เรื่อง:** สิทธิเรียกร้องค่าชดเชยกรณีเลิกจ้างไม่เป็นธรรม

### 1. ข้อเท็จจริง
- ลูกจ้างทำงาน 5 ปี
- นายจ้างเลิกจ้างโดยไม่บอกล่วงหน้า
- ไม่จ่ายค่าชดเชย

### 2. ประเด็นกฎหมาย
1. ลูกจ้างมีสิทธิได้ค่าชดเชยเท่าไร
2. ลูกจ้างมีสิทธิได้ค่าบอกกล่าวล่วงหน้าหรือไม่

### 3. บทบัญญัติที่เกี่ยวข้อง
#### พ.ร.บ. คุ้มครองแรงงาน พ.ศ. 2541
**มาตรา 118** — ค่าชดเชย
- ทำงาน 5 ปี ได้ 10 ค่าจ้างรายสุดท้าย (300 วัน)

**มาตรา 17** — บอกล่วงหน้า
- ต้องบอกล่วงหน้า 1 งวดการจ่ายค่าจ้าง

### 4. วิเคราะห์
- ลูกจ้างมีสิทธิได้ค่าชดเชย 300 วัน
- มีสิทธิได้ค่าบอกกล่าวล่วงหน้า 1 เดือน
- ฟ้องคดีได้ภายใน 1 ปีนับจากผิดนัด

### 5. ข้อเสนอแนะ
1. ส่งหนังสือทวงถามนายจ้าง
2. ร้องเรียนสำนักงานสวัสดิการแรงงาน
3. ฟ้องศาลแรงงานหากไม่ได้รับสิทธิ

⚠️ **Disclaimer:** นี่เป็นข้อมูลเบื้องต้นเท่านั้น
```
