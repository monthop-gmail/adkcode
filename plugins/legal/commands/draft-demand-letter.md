# Command: /draft-demand-letter

## Usage
```
/draft-demand-letter <creditor> <debtor> <amount> <due-date>
```

## Input Parameters
- **creditor:** เจ้าหนี้ (ชื่อ/ที่อยู่)
- **debtor:** ลูกหนี้ (ชื่อ/ที่อยู่)
- **amount:** จำนวนเงิน
- **due-date:** วันครบกำหนดชำระ

## Workflow

### 1. รวบรวมข้อมูล
- ยืนยันข้อเท็จจริง (กู้เมื่อไหร่, สัญญาไหม, มีพยานไหม)
- ตรวจสอบหลักฐาน (IOU, โอนเงิน, สัญญา)
- คำนวณดอกเบี้ย (ถ้ามี)

### 2. ค้นหาบทบัญญัติ
```python
semantic_search("หนังสือทวงถามหนี้ กฎหมาย")
# เจอ: ป.พ.พ. มาตรา 653, 654 (กู้ยืม)
# เจอ: ป.พ.พ. มาตรา 224 (ผิดนัดชำระหนี้)
```

### 3. ร่างหนังสือ
```markdown
[หัวจดหมาย]

ที่ [เลขที่]
[วันที่]

**เรื่อง** หนังสือทวงถามหนี้

**เรียน** [ลูกหนี้]

**อ้างถึง** สัญญากู้ยืมเงินลงวันที่ [date] จำนวนเงิน [amount] บาท

ตามที่ได้มีการทำสัญญากู้ยืมเงินระหว่างท่านกับ [เจ้าหนี้] เมื่อวันที่ [date] 
โดยท่านได้กู้ยืมเงินจำนวน [amount] บาท พร้อมตกลงชำระดอกเบี้ยร้อยละ [X]% ต่อปี 
และกำหนดชำระคืนภายใน [due-date] นั้น

บัดนี้ กำหนดชำระหนี้ได้ล่วงพ้นไปแล้ว แต่ท่านยังมิได้ชำระหนี้ดังกล่าว

ดังนั้น [เจ้าหนี้] จึงขอแจ้งให้ท่านทราบว่าจะต้องชำระหนี้ภายใน [X] วัน 
นับแต่วันที่ได้รับหนังสือฉบับนี้ 

หากท่านมิได้ชำระหนี้ภายในกำหนด [เจ้าหนี้] จะดำเนินคดีตามกฎหมายจนถึงที่สุด 
ซึ่งท่านจะต้องชำระหนี้ต้นพร้อมดอกเบี้ยและค่าใช้จ่ายในการดำเนินคดี

จึงเรียนมาเพื่อโปรดทราบและดำเนินการ

ขอแสดงความนับถือ

[ลายเซ็น]
[ชื่อเจ้าหนี้]

**เอกสารแนบ:**
1. สำเนาสัญญากู้ยืม
2. สำเนาหลักฐานการโอนเงิน
```

### 4. แนะนำขั้นตอนต่อไป
- ส่งทางไปรษณีย์ตอบรับ (สำคัญ!)
- เก็บหลักฐานการส่ง
- หากไม่ชำระ → ฟ้องศาล

## Legal Basis
- **ป.พ.พ. มาตรา 653** — สัญญากู้ยืมเกิน 2,000 บาท ต้องมีหลักฐานเป็นหนังสือ
- **ป.พ.พ. มาตรา 654** — ดอกเบี้ยเกิน 15% ต่อปี เป็นโมฆะ
- **ป.พ.พ. มาตรา 224** — ลูกหนี้ผิดนัดเมื่อถึงกำหนดชำระ
- **ป.วิ.พ. มาตรา 59** — อายุความสัญญาเงินกู้ 10 ปี

## Example
```
> /draft-demand-letter "นาย ก" "นาย ข" 100000 "2025-12-31"

[ร่างหนังสือทวงถามหนี้พร้อมข้อมูลครบถ้วน]

## ขั้นตอนต่อไป
1. พิมพ์หนังสือ พร้อมเซ็นชื่อ
2. ส่งทางไปรษณีย์ตอบรับ (ลงทะเบียนตอบรับ)
3. เก็บหลักฐานการส่ง
4. หากครบกำหนดแล้วไม่ชำระ → ปรึกษาทนายความเพื่อฟ้องคดี

## อายุความ
- ฟ้องคดีได้ภายใน 10 ปีนับจากผิดนัด (ป.พ.พ. 1933)
```
