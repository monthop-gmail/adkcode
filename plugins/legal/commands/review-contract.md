# Command: /review-contract

## Usage
```
/review-contract <path-to-contract>
```

## Workflow

### 1. อ่านสัญญา
```python
read_file(path)
```

### 2. วิเคราะห์โครงสร้าง
- ระบุคู่สัญญา
- วัตถุประสงค์
- ขอบเขตงาน/สิทธิหน้าที่
- ระยะเวลา
- ค่าตอบแทน
- ข้อบอกล่วงหน้า
- ข้อจำกัดความรับผิด
- ข้อระงับข้อพิพาท

### 3. ระบุ Risky Clauses
ใช้ **contract-review skill** จำแนก:
- 🔴 Critical (ต้องแก้)
- 🟡 Warning (ควรพิจารณา)
- 🟢 Suggestion (แนะนำ)

### 4. ค้นหา legal precedents
```python
semantic_search("สัญญาเช่า ข้อควรระวัง")
semantic_search("ข้อจำกัดความรับผิด โมฆะ")
```

### 5. นำเสนอผล
```markdown
## สรุปสัญญา
[ภาพรวม]

## 🔴 ข้อควรระวัง (Critical)
1. Clause X.X — [ปัญหา]
   - **แนะนำ:** [แก้ไขเป็น...]
   - **อ้างอิง:** ป.พ.พ. มาตรา XXX

## 🟡 ข้อควรพิจารณา (Warning)
[รายการ]

## ✅ สิ่งที่ครบถ้วนแล้ว
[รายการ]

## ข้อเสนอแนะ
- ควรแก้ไขก่อนเซ็น: ข้อ 1, 2
- เอกสารเตรียมเพิ่ม: XXX
```

## Example
```
> /review-contract legal/contracts/lease-agreement-draft.md

## สรุปสัญญา
- คู่สัญญา: ผู้ให้เช่า A กับ ผู้เช่า B
- วัตถุประสงค์: เช่าอาคารพาณิชย์
- ระยะเวลา: 3 ปี
- ค่าเช่า: 50,000 บาท/เดือน

## 🔴 ข้อควรระวัง (Critical)
1. Clause 5.3 — ผู้ให้เช่าริบเงินประกันทันทีหากผิดนัด
   - **ปัญหา:** เกินสมควร อาจเป็นโมฆะตาม ป.พ.พ. มาตรา 379
   - **แนะนำ:** แก้เป็น "ริบเงินประกันตามความเสียหายจริง"

## 🟡 ข้อควรพิจารณา (Warning)
1. Clause 8 — ไม่มีข้อระงับข้อพิพาท
   - **แนะนำ:** เพิ่ม "ระงับโดยอนุญาโตตุลาการตามกฎกระทรวง"

## ข้อเสนอแนะ
- แก้ไขข้อ 1 ก่อนเซ็น
- ตรวจสอบโฉนดที่ดินว่าเป็นของผู้ให้เช่าจริง
```
