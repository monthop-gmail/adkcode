# Command: /find-laws

## Usage
```
/find-laws <legal-topic>
```

## Workflow

### 1. ใช้ RAG Semantic Search
```python
# Index กฎหมายไทยก่อน (ถ้ายังไม่ได้ทำ)
index_codebase("legal/thai-laws/")

# ค้นหาด้วยความหมาย
semantic_search("<legal-topic> กฎหมายที่เกี่ยวข้อง")
semantic_search("<legal-topic> มาตรา")
```

### 2. ค้นหาคำพิพากษา
```python
semantic_search("<legal-topic> คำพิพากษา ฎีกา")
web_search("คำพิพากษาฎีกา <legal-topic> ล่าสุด")
```

### 3. ค้นหาจากเว็บ
```python
web_search("พ.ร.บ. <topic> พ.ศ. ล่าสุด")
web_fetch("http://www.krisdika.go.th/law")
```
